# 📦 איך מקבלים קובץ APK להתקנה — בנייה בענן (ללא התקנות במחשב)

המדריך הזה בונה את האפליקציה בענן (חינם) ומפיק קובץ `APK` שמעבירים לטלפון ומתקינים.
לא צריך Flutter, לא Git ולא הרשאות מנהל במחשב — רק דפדפן וחשבון GitHub.

---

## שלב 1 — חשבון GitHub (פעם אחת)

1. היכנסו ל-https://github.com והירשמו (חינם).
2. אשרו את המייל.

## שלב 2 — יצירת מאגר (Repository) חדש

1. לחצו על ה-**+** בפינה הימנית-עליונה ← **New repository**.
2. תנו שם, למשל `wedding-countdown`.
3. בחרו **Public** (חשוב — כך הבנייה בענן חינמית וללא הגבלה).
4. אל תסמנו שום תיבה נוספת. לחצו **Create repository**.

## שלב 3 — העלאת קבצי הפרויקט

1. בעמוד המאגר החדש לחצו על הקישור **"uploading an existing file"**
   (או: לשונית **Add file** ← **Upload files**).
2. פתחו את התיקייה `wedding_countdown` במחשב, בחרו את **כל התוכן** שבתוכה
   (כל הקבצים והתיקיות: `android`, `lib`, `assets`, `.github`, `pubspec.yaml` וכו'),
   וגררו הכול לחלון של GitHub.
   - ודאו שהתיקייה `.github` עלתה (היא מכילה את הוראות הבנייה).
3. המתינו שכל הקבצים יעלו, ואז לחצו **Commit changes**.

## שלב 4 — הפעלת הבנייה

1. עברו ללשונית **Actions** בראש עמוד המאגר.
2. אם מופיעה הודעה לאישור הרצת Workflows — לחצו על הכפתור הירוק לאישור.
3. הבנייה "Build Wedding Countdown APK" תתחיל אוטומטית (אורכת כ-5–10 דקות).
   - אם לא התחילה: בחרו את ה-Workflow בצד שמאל ולחצו **Run workflow**.

## שלב 5 — הורדת ה-APK

1. כשמופיע ✅ ירוק — לחצו על שם הריצה.
2. גללו למטה לקטע **Artifacts**.
3. הורידו את **`wedding-countdown-apk`** (יורד כקובץ ZIP).
4. חלצו את ה-ZIP — בפנים יש קובץ **`app-release.apk`**.

## שלב 6 — התקנה על ה-QIN F22 Pro

1. העבירו את `app-release.apk` לטלפון (כבל USB / וואטסאפ לעצמכם / מייל / Google Drive).
2. בטלפון, פתחו את הקובץ דרך מנהל הקבצים.
3. אם מופיעה אזהרה — אשרו **"התקנה ממקורות לא ידועים / Install unknown apps"**.
4. לחצו **התקן**. סיימתם! 🎉

---

## עדכונים עתידיים
רוצים לשנות משהו (תמונה, שמות וכו')? עדכנו את הקובץ במאגר (Add file ← Upload),
הבנייה תרוץ שוב אוטומטית, והורידו APK חדש מ-Actions.

> טיפ: רוצים APK קטן יותר? אפשר לשנות ב-`.github/workflows/build-apk.yml`
> את שורת הבנייה ל-`flutter build apk --release --split-per-abi` ולהתקין את
> `app-armeabi-v7a-release.apk` (אך ה-APK האוניברסלי הנוכחי מתאים לכל מכשיר).
